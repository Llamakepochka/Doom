Коли перший раз відкриваєш папку з проектом
Дає змогу зберігати різні варіанти коду в різних гілках
git init 

Перед початком встановлення будь якої бібліотеки
Віртуальне середовище
Дає змогу встановлювати бібліотеки суто в рамках проекту
python -m venv .venv(назва папки)

Активація віртуального середовища
source .venv/Scripts/Activate 

Якщо середовище активоване на початку командного рядку буде
(.venv)

Встановлення бібліотеки pygame
pip install pygame 

Створити іншу гілку 
git checkout -b назва_гілки

Перейти назад на гілку master
git checkout master
